SYNC GUARDIAN v0.5.0 — EASIEST WINDOWS BUILD

1. Replace the repository files with this package.
2. Commit and push to GitHub.
3. Open Actions > Build Sync Guardian for Windows.
4. Download the artifact `sync-guardian-0.5.0-windows-x64`.
5. Run `SyncGuardian-Setup.exe`, or extract `SyncGuardian-Portable.zip` into the OBS installation folder.
